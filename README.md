# Inventory Management App

Aplicación web de gestión de inventarios desarrollada como Trabajo de Fin de Grado (TFG).

## Descripción

Este proyecto consiste en el desarrollo de una aplicación orientada a la gestión de inventarios para pequeños y medianos negocios. El objetivo principal es facilitar el control de productos, stock y operaciones básicas de inventario mediante una interfaz sencilla y funcional.

Actualmente, la aplicación cuenta con una base funcional que incluye:

- Conexión a base de datos  
- Operaciones básicas de gestión de productos  
- Sistema de login (provisional)  
- Estructura inicial del proyecto  

## Estado del proyecto

El proyecto se encuentra en desarrollo. A día de hoy:

- Backend funcional con conexión a base de datos  
- Funcionalidades básicas implementadas  
- Interfaz pendiente de mejora (UI/UX)  
- Despliegue en la nube pendiente (actualmente en local)  

## Mejoras futuras

Se plantean las siguientes mejoras para el proyecto:

- Mejora de la interfaz de usuario (diseño, colores y experiencia)  
- Despliegue en la nube para acceso remoto  
- Implementación de funcionalidades avanzadas como:
  - Predicción de pedidos mediante inteligencia artificial  
  - Detección de faltas de stock  
  - Automatización de pedidos a proveedores
